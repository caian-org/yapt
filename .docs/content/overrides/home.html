Welcome to YAPT Documentation

YAPT is an opinionated project template (or scaffold, depending on your background) for Python projects that is constantly evolving. It is a starting point for any kind of project since it doesn't make assumptions about what you're planning to develop or your preferred text editor and operating system.

It can also be used as a reference to already existing projects, since much of it's features (documentation, containerization, pipelines etc) are not interdependent.

The introduction section will give you general information such as YAPT's project structure and what comes out of the box. The quick start will teach you how to run the main example app, the tests, the code quality checks etc. Finally, the API reference section is a showcase on how the python code documentation looks like.
